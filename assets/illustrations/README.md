# 일러스트 플레이스홀더

이 폴더에 활동별 일러스트 이미지를 추가하세요.

## 파일명 규칙

- `technical.png` - 기술 훈련
- `shooting.png` - 슈팅 훈련
- `passing.png` - 패스 훈련
- `pace.png` - 스피드 훈련
- `power.png` - 근력 훈련
- `physical.png` - 체력 훈련
- `mental.png` - 정신력 훈련
- `defending.png` - 수비 훈련
- `balanced.png` - 종합 훈련
- `rest.png` - 휴식
- `go_out.png` - 외출

## 추천 사양

- **크기**: 800×600px (4:3 비율)
- **형식**: PNG (투명 배경 지원)
- **스타일**: 애니메이션 느낌의 일러스트

## 현재 상태

현재 모든 일러스트는 색상 플레이스홀더로 표시됩니다.
실제 이미지 파일을 추가하면 자동으로 로드됩니다.
